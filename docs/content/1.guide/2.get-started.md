```ts [style.config.ts]

```

ğŸ¨. [**Use your tokens!**]{.text-xl}

```vue [layout/default.vue]

 </header>
</main>
</template>

<style lang="postcss" scoped>
/* <style> template usage */

  background-color: v-bind($tokens('colors.secondary'))

</style>
```
